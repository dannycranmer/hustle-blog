## What is it?

A daily content site serving up interesting, random stories — discovered and curated by AI. Every visit surfaces something new. Hit shuffle, learn something unexpected.

**Why it exists:** Content diversifies the portfolio beyond utility tools. Different audience, different engagement pattern, different monetisation path.

## How it works

- AI discovers and writes bite-sized interesting stories daily
- React SPA with a clean, fast reading experience
- One-tap shuffle for the next story
- Hosted on Netlify (free tier)

## Tech Stack

React SPA built with Vite. Deployed on Netlify. GoatCounter analytics.
