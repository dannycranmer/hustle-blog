## What is it?

A free suite of 28 online developer utilities. No ads, no tracking, no clutter — just tools that work. Everything runs 100% client-side in your browser. Your data never leaves your machine.

**Why it exists:** Competitors like jsonformatter.org get millions of monthly visits but are ugly, ad-riddled, and send your data to their servers. DevBrew is the clean, private alternative.

## Tools (28)

**Data Formats:** JSON Formatter, Base64, URL Encoder, JWT Decoder, YAML/JSON, XML/JSON, CSV Viewer, HTML Entity, SVG Optimizer

**Code Tools:** Regex Tester, Diff Checker, CSS Minifier, JS Minifier, Markdown Preview, SQL Formatter

**Generators:** UUID, Password, Lorem Ipsum, Hash, Favicon, Chart, OG Image, Gradient, QR Code

**Utilities:** Cron Parser, Unix Timestamp, Color Picker, Unit Converter

## Tech Stack

Vanilla HTML/CSS/JS. Zero dependencies, zero build step. Deployed on GitHub Pages.
