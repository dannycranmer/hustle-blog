## Step-by-Step Guide

### Step 1: Go to Hacker News

Open **https://news.ycombinator.com/** in your browser. Log in (or create an account if you don't have one).

### Step 2: Click "submit" in the top nav

You'll see a link that says "submit" in the top navigation bar. Click it.

### Step 3: Fill in the submission

**Title** (copy exactly):
```
Show HN: Parchment – 17 free PDF tools that run entirely in your browser
```

**URL:**
```
https://parchpdf.com/
```

**Leave the "text" field empty** — for Show HN with a URL, the URL is the main content.

### Step 4: Submit

Click the submit button. Your post is now live on the "newest" page.

### Step 5: Add a first comment (important!)

Immediately after posting, add a comment on your own post. This is standard for Show HN. Copy this or use the draft:

> Hi HN! I built Parchment as a privacy-first alternative to tools like iLovePDF and SmallPDF. Everything runs client-side using pdf-lib and PDF.js — your files never leave your browser.
>
> 17 tools: merge, split, compress, convert, rotate, unlock, protect, watermark, page numbers, PDF-to-image, image-to-PDF, sign, redact, flatten, crop, extract, and a comparison page.
>
> It's free, no signup required, no file size limits. Source: [GitHub link]
>
> Happy to answer any technical questions about the client-side approach!
